# 👨‍💻 Hi, I'm Masuda — 業務を深く理解して自動化を設計・構築できるエンジニア

人事部門に常駐し、VBAによる業務支援ツールを**35件以上**単独で開発・保守してきた実務特化型エンジニアです。

仕様書のない業務でも、ヒアリングから要件を引き出し、Excel・Outlook・WinAPI・Web操作を駆使して現場業務を徹底的に自動化してきました。

また、「高速に動作し、使い心地の良いツール」を常に意識しており、マクロの動作遅延や操作ストレスといった"VBAあるある"を解消する設計に注力。既存マクロの解析・高速化・再設計も得意としています。

現在はPowerAutomate・TypeScript・AWSなどのクラウド・モダン技術を習得中で、「**既存のExcel業務をクラウド上で再設計・再構築する**」を軸に、2026年にDX支援型のフリーランスとしての独立を目指しています。

> 💡 **2025年実績**: 1年間で新規ツール20件以上を開発、人事システムやインフラ更改対応、継続保守・改善も並行実施

## 🤖 AI駆動開発オーケストレーション（2025〜現在）

**15件以上のプロジェクトをAIエージェントと並行開発**する独自のオーケストレーションシステムを構築・運用中。

### 活用しているAI機能

| 機能 | 用途 |
|------|------|
| **Agents** | 複数のサブエージェントによる並列タスク実行（調査/実装/テスト/ログ記録の分離） |
| **Skills** | プロジェクト固有の定型操作をスキル化（Bot管理/セッション初期化/パス解決） |
| **Hooks** | セッション開始・終了時の自動処理（コンテキストロード/作業ログ自動振分け） |
| **Rules** | ドメイン別のコーディング規約・制約の自動適用（Rust/戦略/取引所別） |
| **MCP** | 外部ツール・データソースとの連携拡張 |

### オーケストレーションの特徴

- **階層型ガバナンス**: ルート→カテゴリ→プロジェクト→モジュールの4階層でCLAUDE.mdを継承
- **Bot別ログ管理**: 作業履歴をBot/機能別に自動振り分け、コンテキスト汚染を防止
- **モデル使い分け**: 計画フェーズはOpus、実装はSonnet、ログ記録はHaikuで最適化
- **セッション継続性**: 中断しても前回の作業状態から即座に再開可能
